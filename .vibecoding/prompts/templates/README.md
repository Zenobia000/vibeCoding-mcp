## 🔧 使用方式

1. **選擇適當模板**: 根據 AI 服務類型選擇對應模板
2. **替換變數內容**: 將 `{{ 變數名稱 }}` 替換為實際內容
3. **調整具體內容**: 根據項目需求調整章節和詳細內容
4. **確保品質標準**: 遵循每個模板的品質標準要求
