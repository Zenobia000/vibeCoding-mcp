**解決**: 修正為正確的源碼路徑 `../../src/`
**影響**: 解決編譯失敗和循環依賴問題

#### 2. 類型安全增強
**新增**: MCP 參數類型轉換輔助函數
- `extractParam<T>()` - 通用參數提取
- `extractStringParam()` - 字符串參數提取
- `extractOptionalStringParam()` - 可選字符串參數
- `extractArrayParam<T>()` - 陣列參數提取
- `extractObjectParam<T>()` - 物件參數提取
